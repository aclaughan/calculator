# calculator
As the name suggests will perform calculations on two numbers. After each calculation you can either perform another operation on the previous answer or start a new calculation. Of cause if you've had enough goodness you can always press 'q' to exit :)

```text
 _____________________
|  _________________  |
| | Pythonista   0. | |  .----------------.  .----------------.  .----------------.  .----------------. 
| |_________________| | | .--------------. || .--------------. || .--------------. || .--------------. |
|  ___ ___ ___   ___  | | |     ______   | || |      __      | || |   _____      | || |     ______   | |
| | 7 | 8 | 9 | | + | | | |   .' ___  |  | || |     /  \     | || |  |_   _|     | || |   .' ___  |  | |
| |___|___|___| |___| | | |  / .'   \_|  | || |    / /\ \    | || |    | |       | || |  / .'   \_|  | |
| | 4 | 5 | 6 | | - | | | |  | |         | || |   / ____ \   | || |    | |   _   | || |  | |         | |
| |___|___|___| |___| | | |  \ `.___.'\  | || | _/ /    \ \_ | || |   _| |__/ |  | || |  \ `.___.'\  | |
| | 1 | 2 | 3 | | x | | | |   `._____.'  | || ||____|  |____|| || |  |________|  | || |   `._____.'  | |
| |___|___|___| |___| | | |              | || |              | || |              | || |              | |
| | . | 0 | = | | / | | | '--------------' || '--------------' || '--------------' || '--------------' |
| |___|___|___| |___| |  '----------------'  '----------------'  '----------------'  '----------------' 
|_____________________|

Input your calculation. e.g. 23 * 16
  > 42 * 17
42 * 17 = 714.0
press 'y' to continue calculating
press 'n' for a new calculation
or press 'q' to quit,
  > y
Input your calculation. e.g. + 32
  > + 26
714.0 + 26 = 740.0
press 'y' to continue calculating
press 'n' for a new calculation
or press 'q' to quit,
  > y
Input your calculation. e.g. + 32
  > / 36
714.0 / 36 = 19.833333333333332
press 'y' to continue calculating
press 'n' for a new calculation
or press 'q' to quit,
  > n
Input your calculation. e.g. 23 * 16
  > 23 * 16
23 * 16 = 368.0
press 'y' to continue calculating
press 'n' for a new calculation
or press 'q' to quit,
  > y
Input your calculation. e.g. + 32
  > + 32
368.0 + 32 = 400.0
press 'y' to continue calculating
press 'n' for a new calculation
or press 'q' to quit,
  > q
```
